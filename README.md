# Quizora - Frontend Client

## 📖 Project Overview
A modern, responsive frontend application built with **Next.js**, designed to function as a standalone client within a **decoupled full-stack architecture**. This project demonstrates the ability to transform complex **Figma** designs into high-performance interfaces that communicate securely with a remote **Express.js API** across different origins.

## 🛠 Tech Stack
* **Framework:** Next.js (React)
* **Testing:** Jest
* **Styling:** Tailwind CSS / CSS Modules
* **Deployment:** Vercel

## 🚀 Key Features
* **Design Fidelity:** Adapted and implemented UI components based on community designs by **Olowu 001** on Figma.
* **Cross-Origin Integration:** Engineered to perform secure, asynchronous data fetching from a separate Express.js backend, handling **CORS** and complex state synchronization.
* **Testing Suite:** Ensured reliability and component stability with comprehensive **unit tests** authored in **Jest**.
* **Deployment Approach:** Hosted on **Vercel** with a managed CI/CD pipeline, utilizing environment variables to securely connect to the production API.
